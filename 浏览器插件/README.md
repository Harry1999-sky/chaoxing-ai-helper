# 超星学习通自动搜题助手

一个 Chrome/Edge 浏览器插件，用于自动识别超星学习通题目并搜索答案。

## 功能特性

- ✅ 自动识别页面上的题目（单选、多选、判断、填空）
- ✅ 多个搜题 API 接口，自动切换
- ✅ 自动填写答案
- ✅ 浮动控制面板，可拖拽
- ✅ 右键搜索选中文本
- ✅ 答题统计
- ✅ 自定义 API 接口

## 安装步骤

### 1. 生成图标

1. 用浏览器打开 `generate-icons.html`
2. 点击下载按钮，分别下载 16x16、48x48、128x128 三个图标
3. 将下载的图标保存到 `icons` 文件夹中

### 2. 安装到浏览器

#### Chrome 浏览器
1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹

#### Edge 浏览器
1. 打开 Edge，地址栏输入 `edge://extensions/`
2. 开启左侧的「开发人员模式」
3. 点击「加载解压缩」
4. 选择本项目文件夹

### 3. 使用方法

1. 打开超星学习通的作业或考试页面
2. 点击浏览器工具栏上的插件图标
3. 点击「扫描题目」查看识别结果
4. 点击「开始」自动搜题答题
5. 页面右侧会出现浮动控制面板

## 配置说明

### 基本设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 自动填写 | 找到答案后自动填写 | 开启 |
| 自动提交 | 填写完自动提交（谨慎使用） | 关闭 |
| 显示浮动面板 | 在页面上显示控制面板 | 开启 |

### 搜题接口

- **题库网**: 默认接口，覆盖较广
- **小猿题库**: 备用接口
- **自定义**: 可配置自己的 API

### 答题间隔

- 范围: 500ms - 5000ms
- 默认: 1500ms
- 建议不要设置太短，避免被检测

## 常见问题

### Q: 扫描不到题目？
A: 请确保页面已完全加载，尝试刷新页面后重新扫描。

### Q: 搜不到答案？
A: 可能是题库中没有收录该题目，尝试切换 API 接口。

### Q: 答案填写不正确？
A: 自动识别可能有误，建议手动检查后再提交。

### Q: 被检测到怎么办？
A: 增大答题间隔，不要使用自动提交功能。

## 开发说明

如需修改或扩展功能：

1. `content.js` - 核心搜题和填写逻辑
2. `popup.js` - 弹出窗口设置界面
3. `background.js` - 后台数据处理

### 添加新的搜题 API

在 `content.js` 的 `CONFIG.apis` 数组中添加新接口：

```javascript
{
  name: '接口名称',
  url: 'https://api.example.com/search',
  method: 'POST',
  buildBody: (question) => `q=${encodeURIComponent(question)}`,
  headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
  parse: (response) => {
    // 解析响应，返回答案字符串
    const data = JSON.parse(response);
    return data.answer || '';
  }
}
```
